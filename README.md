# AI Product Manager Vision Demo

This repository contains a guided interactive simulation
showing a future AI-native day in the life of a product manager.

The demo focuses on:
- AI product signal orchestration
- hypothesis generation
- experiment simulation
- automated artefact creation
- accelerated decision cycles

This is a storytelling prototype, not a production system.

## Stack

- Next.js App Router
- React
- Tailwind CSS
- Framer Motion

## Run

```bash
npm install
npm run dev
```
